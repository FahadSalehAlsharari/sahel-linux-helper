# سهل — مدير النظام العربي للينكس

**سهل** هو أداة صيانة وتحديث احترافية تعمل داخل الطرفية، صُممت لمستخدمي أوبنتو وديبيان، بواجهة عربية واضحة ومباشرة.

الهدف بسيط: تجميع مهام الصيانة اليومية في مكان واحد، بطريقة آمنة ومفهومة، بدون ما يحتاج المستخدم يحفظ أوامر طويلة أو يخاطر بتنفيذ شيء غلط.

---

## لمن هذه الأداة؟

- المستخدمين الجدد في لينكس، خصوصًا القادمين من ويندوز
- من يبحث عن طريقة أسهل لصيانة النظام من الطرفية
- من يريد أداة عربية عملية وخفيفة بدون تعقيد
- المستخدمين المتقدمين اللي يبون أداة سريعة للمهام الروتينية

---

## المزايا

- **صيانة آمنة** — تحديث + ترقية + تنظيف بضغطة واحدة
- **معاينة قبل التنفيذ** — شوف وش راح يتغير بدون ما يتعدل شيء (Dry Run)
- **صيانة متقدمة** — ترقية كاملة عبر dist-upgrade مع تحذيرات واضحة
- **تنظيف فقط** — تنظيف الحزم والملفات المؤقتة بدون إنترنت
- **حالة النظام** — عرض شامل للنظام، مديري الحزم، واستخدام القرص
- دعم **APT** و **Snap** و **Flatpak**
- فحوصات أمان قبل أي عملية
- ملخص واضح بعد كل عملية
- تسجيل كامل للعمليات في ملفات log
- واجهة تفاعلية عربية + أوامر CLI مباشرة

---

## الأنظمة المدعومة

- **Ubuntu** (20.04 وأعلى)
- **Debian**
- الأنظمة المبنية عليهما

> يتطلب Bash 4.4 أو أعلى.

---

## التثبيت

```bash
git clone https://github.com/FahadSalehAlsharari/sahel-linux-helper.git
cd sahel-linux-helper
chmod +x sahel
```

### تثبيت كأمر في النظام

```bash
sudo install -m 755 sahel /usr/local/bin/sahel
```

بعدها تقدر تشغلها مباشرة من أي مكان:

```bash
sahel
```

## طريقة الاستخدام

### الوضع التفاعلي

شغّل الأداة بدون أي خيارات وراح تظهر لك قائمة واضحة:

```bash
sudo sahel
```

### الأوامر المباشرة

```bash
sudo sahel --safe          # صيانة آمنة (تحديث + ترقية + تنظيف)
sudo sahel --preview       # معاينة بدون تنفيذ
sudo sahel --advanced      # صيانة متقدمة (dist-upgrade)
sudo sahel --cleanup       # تنظيف فقط
sudo sahel --status        # عرض حالة النظام
sahel --help               # عرض المساعدة
sahel --about              # حول الأداة
```

---

## الأمان

الأداة صُممت بعقلية أمنية:

- فحوصات تلقائية قبل أي عملية (التوزيعة، المساحة، الاتصال، أقفال dpkg)
- لا تقوم بترقية إصدار النظام ولا تعدّل المصادر
- تحذيرات واضحة قبل العمليات المتقدمة مع طلب تأكيد
- تسجيل كامل لجميع العمليات في `/var/log/`

> مع ذلك، بعض العمليات بطبيعتها تحتاج حذر. دائمًا تأكد إنك فاهم وش راح يصير قبل ما توافق.
