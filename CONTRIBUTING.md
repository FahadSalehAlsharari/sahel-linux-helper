# المساهمة في سهل

شكرًا لمساهمتك في مشروع **سهل**.

## مبادئ المشروع

- الحفاظ على هوية المشروع العربية: **سهل**.
- الحفاظ على ملكية ونَسب Fahad Alsharari بالكامل.
- إبقاء الأداة Bash-first وموجهة لأوبنتو سطح المكتب.
- تقديم سلوك آمن افتراضيًا وتقارير نهائية صادقة.

## إرشادات التطوير

- لا تضف ميزات خارج نطاق الصيانة المعتادة للنظام.
- تجنب أي سلوك تدميري افتراضيًا.
- حافظ على بساطة الواجهة ووضوح الرسائل العربية.
- تأكد أن العدادات النهائية متسقة دائمًا مع النتائج الفعلية.

## قبل فتح Pull Request

```bash
bash -n sahel
```

واختبر المسارات الأساسية عند الإمكان:

```bash
sudo ./sahel --preview
sudo ./sahel --safe
sudo ./sahel --cleanup
```

- حدّث `CHANGELOG.md` عند وجود تغييرات سلوكية.
- راجع `README.md` لضمان تطابقه مع السلوك الحقيقي.

## جودة PR

- وصف واضح للمشكلة والحل.
- توضيح أي أثر على الأمان أو السلوك.
- إرفاق مخرجات اختبار مختصرة عند الحاجة.
