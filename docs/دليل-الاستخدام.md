# دليل الاستخدام

يوضح هذا الملف طريقة استخدام أداة **سهل** سواء عبر الواجهة التفاعلية أو عبر الأوامر السريعة من الطرفية.

---

## تشغيل الأداة

يمكن تشغيل **سهل** بطريقتين:

### التشغيل من داخل مجلد المشروع

```bash
./sahel
```

### أو بعد تثبيتها داخل النظام

```bash
sahel
```

---

## أسلوب الاستخدام

يدعم المشروع نمطين أساسيين للاستخدام:

### 1) الواجهة التفاعلية
هذا هو الوضع الأنسب للمستخدم المبتدئ، حيث تظهر قائمة عربية واضحة داخل الطرفية تتيح لك التنقل بين الوظائف المختلفة بسهولة.

### 2) الأوامر السريعة
هذا الوضع مناسب عندما تريد تنفيذ مهمة معينة مباشرة دون الدخول إلى الواجهة التفاعلية.

---

## الأوامر السريعة

### عرض المساعدة

```bash
sahel --help
```

يعرض هذا الأمر شرحًا مختصرًا للخيارات المتاحة وطريقة الاستخدام.

---

### عرض معلومات النظام

```bash
sahel --info
```

يعرض هذا الأمر معلومات أساسية عن الجهاز والنظام، مثل:
- اسم النظام
- اسم الجهاز
- إصدار النواة
- مدة التشغيل
- الذاكرة
- مساحة التخزين الأساسية

---

### تحديث النظام

```bash
sahel --update
```

يستخدم هذا الأمر لتنفيذ تحديث أساسي للحزم والخدمات المدعومة وفق ما تدعمه النسخة الحالية من الأداة.

---

### تنظيف النظام

```bash
sahel --clean
```

ينفذ هذا الأمر تنظيفًا آمنًا للحزم غير الضرورية وبعض الملفات المؤقتة المرتبطة بإدارة الحزم.

---

## أمثلة استخدام

### مثال 1: تشغيل الواجهة العربية

```bash
sahel
```

### مثال 2: عرض معلومات النظام بسرعة

```bash
sahel --info
```

### مثال 3: تنفيذ تحديث مباشر

```bash
sahel --update
```

### مثال 4: تنفيذ تنظيف مباشر

```bash
sahel --clean
```

---

## متى أستخدم الواجهة التفاعلية؟

يفضل استخدام الواجهة التفاعلية في الحالات التالية:

- إذا كنت مستخدمًا جديدًا في لينكس
- إذا كنت لا تحفظ الأوامر
- إذا كنت تريد رؤية الخيارات بشكل مرتب
- إذا كنت تريد تقليل احتمال الخطأ في التنفيذ

---

## متى أستخدم الأوامر السريعة؟

يفضل استخدام الأوامر السريعة في الحالات التالية:

- إذا كنت تعرف المهمة التي تريد تنفيذها مباشرة
- إذا كنت تريد سرعة أكبر
- إذا كنت تستخدم الأداة بشكل متكرر
- إذا كنت تدمجها لاحقًا في اختصارات أو سكربتات أخرى

---

## ملاحظات مهمة أثناء الاستخدام

- بعض المهام تتطلب صلاحيات `sudo`.
- حذف الحزم يجب أن يتم بحذر.
- الأداة موجهة للتسهيل، لكنها لا تلغي الحاجة للفهم الأساسي لما تفعله.
- يفضل عدم تنفيذ أي حذف إلا عند معرفة أثره على النظام.

---

## نصيحة للمستخدم الجديد

إذا كنت تستخدم لينكس لأول مرة، فابدأ بالواجهة التفاعلية أولًا، لأنها أوضح وأسهل، ثم انتقل بعد ذلك إلى الأوامر السريعة عندما تصبح أكثر اعتيادًا على بيئة النظام.

---

## الهدف من هذه الأداة

الهدف من **سهل** ليس استبدال فهم لينكس، بل تسهيل البداية، وتقليل الحيرة، وتقديم تجربة عربية أوضح داخل الطرفية.
