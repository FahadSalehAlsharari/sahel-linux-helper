# دليل التثبيت

يوضح هذا الملف طرق تثبيت وتشغيل أداة **سهل** على الأنظمة المدعومة، مع الحفاظ على البساطة والوضوح للمستخدم العربي.

---

## الأنظمة المدعومة

الإصدار الحالي من **سهل** موجه بشكل أساسي إلى:

- Ubuntu
- Debian
- الأنظمة المبنية عليهما

---

## المتطلبات

قبل تشغيل الأداة، تأكد من توفر التالي:

- نظام لينكس مدعوم
- Bash
- طرفية Terminal
- صلاحيات `sudo` عند الحاجة
- اتصال بالإنترنت في حال تنفيذ التحديثات أو تثبيت الحزم

---

## الطريقة الأولى: التشغيل المباشر من داخل المشروع

إذا كان لديك ملف الأداة داخل مجلد المشروع، يمكنك تشغيله مباشرة بهذه الخطوات:

### 1) الدخول إلى مجلد المشروع

```bash
cd ~/Desktop/FahadScript
```

### 2) إعطاء الملف صلاحية التنفيذ

```bash
chmod +x sahel
```

### 3) تشغيل الأداة

```bash
./sahel
```

---

## الطريقة الثانية: تثبيت الأداة كأمر دائم في النظام

إذا أردت تشغيل الأداة من أي مكان في النظام دون الحاجة للدخول إلى مجلد المشروع في كل مرة، يمكنك تثبيتها يدويًا داخل:

```bash
/usr/local/bin
```

### أمر التثبيت

```bash
sudo install -m 755 sahel /usr/local/bin/sahel
```

بعد ذلك يمكنك تشغيلها مباشرة عبر:

```bash
sahel
```

---

## التحقق من نجاح التثبيت

بعد التثبيت، جرّب أحد الأوامر التالية:

```bash
sahel --help
```

أو:

```bash
sahel --info
```

إذا ظهرت المساعدة أو معلومات النظام بشكل طبيعي، فهذا يعني أن التثبيت تم بنجاح.

---

## تحديث الأداة بعد تعديلها

إذا قمت بتعديل ملف `sahel` داخل المشروع، وأردت استبدال النسخة المثبتة في النظام، أعد تنفيذ هذا الأمر:

```bash
sudo install -m 755 sahel /usr/local/bin/sahel
```

---

## إزالة الأداة من النظام

إذا كنت قد ثبت الأداة يدويًا داخل `/usr/local/bin`، يمكنك حذفها عبر:

```bash
sudo rm -f /usr/local/bin/sahel
```

---

## ملاحظات مهمة

- بعض العمليات داخل الأداة تحتاج صلاحيات إدارية.
- يفضل اختبار الأداة أولًا بشكل مباشر قبل تثبيتها على مستوى النظام.
- إن كنت تطور المشروع باستمرار، فالتشغيل من داخل مجلد المشروع يكون مناسبًا في البداية.
- عند نشر نسخ أحدث لاحقًا، يمكن التفكير بإضافة مثبت Installer أكثر تنظيمًا.

---

## توصية عملية

إذا كنت ما زلت في مرحلة التطوير والتجربة، فاستخدم:

```bash
./sahel
```

أما إذا أردت استخدام الأداة بشكل يومي وكأنها أمر نظام عادي، فاستخدم:

```bash
sudo install -m 755 sahel /usr/local/bin/sahel
```

ثم شغلها بالأمر:

```bash
sahel
```
